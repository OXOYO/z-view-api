# z-view-api

[z-view](https://github.com/OXOYO/z-view) 配套 API 服务。

[![node](https://img.shields.io/badge/node-v7.6.0+-blue.svg)](https://nodejs.org/)

## 开始
```bash
  git clone
  npm i
  npm run dev
```
Node.js 版本需 >= 7.6.0.

## 打包
```bash
  npm run build
```

## 生产环境
```bash
  pm2 start ecosystem.config.js
```
